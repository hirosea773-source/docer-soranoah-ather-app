バズランクについて

バズランクは、YouTubeやSNSで今この瞬間に話題となっている動画を独自のアルゴリズムで解析し、ランキング形式でお届けするトレンド情報サイトです。再生回数だけでなく、SNSでの拡散数、注目度の急上昇率などを多角的に分析。音楽、ゲーム、テック、ニュースなど、幅広いジャンルの「今見るべき動画」を一目で把握できます。

よくある質問

ランキングはいつ更新されますか？

ランキングは毎日12:00と0:00の2回、最新のデータを集計して更新されます。

ランキングの集計基準は何ですか？

直近24時間の再生回数、高評価数、コメント数に加え、主要SNSでの共有数などを独自のウェイトでスコアリングしています。

動画の掲載を依頼できますか？

現在は自動収集のみとなっており、個別のご依頼は承っておりませんが、広告枠での掲載についてはお問い合わせフォームよりご連絡ください。
